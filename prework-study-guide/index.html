Prework Study Guide

HTML

  • Hypertext Markup Language
  • Handles content of a website as well as its structure
    and meaning of each element on the page.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • Cascading Style Sheets
  • Language used to describe the presentation of a website that's
    created with HTML and how each element on a webpage is rendered.

Git

  • A free, open source version control system that allows
    web developers to collaborate on projects without overwriting
    each other's work.
  • Also allows developers to revert to previous versions of the code
    if a probblem is discovered.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add: to stage a file (ready for commit)
  • git commit: to commit to repository (ready for merge).

JavaScript

  • A lightweight programming language that is used to
    provide logic & interactivity for websites.
